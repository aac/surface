# Security Policy

## Reporting a vulnerability

Found a security issue in `surface`? Please report it privately using the **Report a vulnerability** button on the **Security** tab of this repository, rather than opening a public issue.

`surface` is maintained by one person. Reports are genuinely welcome and I'll look into them.

This repo is a skill bundle plus reference implementations; the deeper security model for agent-owned surfaces lives in [`skills/surface/references/security.md`](skills/surface/references/security.md).
